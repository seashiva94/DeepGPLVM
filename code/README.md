### Representation Learning Using Gaussian Processes

The code is in the ipython notebook as the model was trained on
Google Colaboratory

The necessary classes and helper functions are in the utils.py file which is
used inside the notebook.

The dependencies include:

gpflow
numpy
tensorflow
pytorch and torchvision (to download MNIST dataset easily)


The data folder consists of the csv files for the oil-flow datset that is
used for the project.
